A different world map:

Using a geometric interpretation of data, a high-dimensional space is created from data about countries. Each country is a point in space and each metric is a dimension.

The points in high-dimensional space are projected and embedded in two-dimensions using dimensionality reduction, clustering and topological data analysis. The result is a map that tries to preserve the structure in high-dimensional space.

A new world map with different borders and regions. A different geography of people, lives and ties.

Many thanks to the people that created the open-source tools and data sources used in this project
