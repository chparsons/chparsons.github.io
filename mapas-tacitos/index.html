A different world map from a geometric interpretation of data:

A high-dimensional space is created from socio-economic data about countries. Each country is a point in space, each metric is a dimension.

Using dimensionality reduction, clustering and topological data analysis, the points in high-dimensional space are projected and embedded in two-dimensions. The result is a map that approximates the shape of the data in high-dimensional space.

A new world map with different borders and regions. A geography of people, lives and ties.
