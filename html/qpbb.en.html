Tangible interface for Map.


The project explores data visualization related to:

» tangible thinking

» collaboration

» parallel human-human interaction


The map displays environmental data of the petrochemical center of Bahía Blanca: plants discharge effluents on the water. Each circle is a measurement of a effluent at a point in time. The radius is mapped to the deviation of the discharged substance with respect to its legal limit. The inner colored cirle on each plant shows the proportion of substances discharged within the time range selected by users.

see http://www.quepasabahiablanca.gov.ar/ for a web interface using the same data
